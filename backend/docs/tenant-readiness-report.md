# Tenant Readiness — Relatório Pilot

**Motores:** `tenantOperationalComplexityScoring` · `tenantReadinessClassifier` · `enterprisePilotTenantSelector`

## Classificação

| Nível | Critério resumido |
|-------|------------------|
| LOW | Score operacional baixo |
| MODERATE | Complexidade ou score médio |
| ADVANCED | Publication estável + score ≥58 |
| INDUSTRIAL_READY | Score ≥72, friction ok, sem overload |

## Selecção

Máximo de tenants piloto configurável (`max_pilots`). Sem auto-promoção.
