# Ecosystem Correlation Runtime

**Camada:** `backend/src/ecosystem-correlation/`  
**API:** `POST /api/ecosystem-correlation/pack`

Consolida correlação cross-domain do ecossistema IMPETUS (Q · S · L · E · produção · manutenção · energia).

## Runtimes

- `ecosystemCorrelationRuntime` — orquestrador principal
- `ecosystemCrossDomainRuntime` — pares de domínio
- `ecosystemTelemetryCorrelationRuntime` — fusão telemétrica
- `ecosystemExecutiveCorrelationRuntime` — narrativas e heatmaps

## Regras

- `assistive_only: true`
- `auto_promotion: false`
- `shadow_first: true`
