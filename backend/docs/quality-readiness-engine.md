# Readiness engine (rollout)

## Motor

`qualityIndustrialReadinessEngine.js` — checklist (telemetria WAVE3, governança qualidade, V3 storage, eventos industriais, inputs de utilizador).

## Saídas

`readiness_score`, `readiness_blockers`, `rollout_recommendation`, `safe_activation_hints`.

## Evento

`quality.rollout.readiness_blocked` quando há blockers de readiness declarados.
