# Centro de Inteligência Industrial — Prompt Definitivo v3.0

## 🎯 Objetivo Central

Substituir o dashboard fixo e igual para todos os usuários por um sistema dinâmico onde a IA monta automaticamente a interface correta para cada cargo, setor e função — transformando o IMPETUS em um Centro de Comando Industrial Inteligente.

## Escopo de implementação (Cursor)

- **Alterar apenas:** páginas/componentes/rotas e APIs do **dashboard** (grid dinâmico, widgets, centros, Cérebro Operacional no contexto do dashboard).
- **Não alterar:** módulos independentes (Cadastrar com IA, Pró-Ação, Registro com IA, Chat Impetus, Impetus IA, Validação Organizacional, Configurações, etc.); quando o prompt citar esses módulos, usar apenas no contexto do dashboard (links, resumos, redirecionamentos) sem modificar o código deles.
